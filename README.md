# 💖 LoveSite — сайт-поздравление с Днём Рождения

Романтичный одностраничный сайт-сюрприз для любимого человека. Никаких фреймворков и сборщиков — только чистые **HTML, CSS и JavaScript**. Открывается в любом браузере, работает на телефоне и компьютере.

## ✨ Что внутри

- 🎵 **Фоновая музыка** — стартует по кнопке на экране приветствия, кнопка-пластинка в углу для паузы
- 💗 **Летающие сердечки** — анимированный фон на canvas по всему сайту
- ⌨️ **Печатающиеся фразы** — признания появляются по буквам на главном экране
- ⏱️ **Счётчик отношений** — дни, часы, минуты и секунды вместе в реальном времени
- 🌹 **6 карточек** «Почему я тебя люблю»
- 📸 **Галерея** — фото и видео с просмотром на весь экран, видео проигрываются при наведении
- 💌 **Конверт с письмом** — открывается по клику, текст печатается по буквам
- 🎉 **Финальная кнопка** — взрыв конфетти и сердечек
- 📱 Полностью адаптивный дизайн

## 🚀 Как запустить

Просто откройте `index.html` в браузере — ничего устанавливать не нужно.

```bash
git clone https://github.com/Nurbolottop/LoveSite.git
cd LoveSite
open index.html
```

## ⚙️ Как настроить под себя

Все настройки — в начале файла [`script.js`](script.js) в блоке `CONFIG`:

```js
const CONFIG = {
  // Имя любимой (показывается на главном экране)
  name: 'Любимая',

  // Дата начала отношений: год, месяц (0 = январь!), день
  relationshipStart: new Date(2025, 5, 11),

  // Фразы, которые печатаются на главном экране
  typedPhrases: [ ... ],

  // Текст письма в конверте
  letterText: ` ... `,
};
```

Свои фото, видео и музыку положите в папку `images_videos/` и укажите пути в `index.html`.

## 📁 Структура проекта

```
LoveSite/
├── index.html      — разметка всех секций
├── style.css       — стили и анимации
├── script.js       — логика: музыка, счётчик, галерея, конфетти
└── images_videos/  — фото, видео и музыка
```

## 🛠️ Технологии

- HTML5, CSS3 (анимации, grid, backdrop-filter)
- Ванильный JavaScript (Canvas API, IntersectionObserver)
- Google Fonts: Marck Script, Playfair Display, Montserrat
